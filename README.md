# SIGNAL — Capture the Signals

Champion Digital Media's playable intent-data pitch: buyer intent fires across four
freight modes (plane / ship / truck / train — ad, email, website, and news signals).
Pilot one mode at a time and capture every signal before it goes cold.

Single-file Three.js game. Deployed automatically to GitHub Pages on push to `main`.

**Play:** https://jaredcdm.github.io/signal_hunter/
